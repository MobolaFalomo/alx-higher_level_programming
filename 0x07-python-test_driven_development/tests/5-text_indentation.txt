#5-text_indentation

==================================
HOW TO USE THE 5-text_indentation
==================================

This module defines a function ''text_indentation(text)'' that
prints a text with two new lines after each of the characters
'.', '?' and ':'


USAGE
=====

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("""Lorem ipsum dolor sit amet, consectetur adipiscing elit.
